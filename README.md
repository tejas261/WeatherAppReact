This is a Weather App that I built to know the weather conditions of a place using React.
The following details of the city are being provided by this application:
1. Temperature
2. Current Weather condition
3. Humidity
4. Wind Speed

As mentioned, this app is built using React, a frontend JavaScript framework and JSX which is a mixture of HTML and JavaScript.
Also an external API is used that is provided by OpenWeather to fetch the weather details of any location

Download Link : https://github.com/tejas261/WeatherAppReact

Steps to  run this app on your system locally:

1. Install Node.js globally on your system

2. Place the downloaded folder in your desired directory.

3. Open any code editor preferably VS Code and open the folder where you have place dthe downloaded file.

4. Now open terminal in VS Code and change the directory to the weatherAppReact directory.

5. Type 'npm start' in the terminal and wait for the app to launch.

6. Once the app is launched you would see a searchbar wehre you have to enter the name of the city whose weather details are to be gathered.

7. Make sure to not give a whitespace after the city name and click on the search button.

8. You will  get the details of that loaction on your screen.
